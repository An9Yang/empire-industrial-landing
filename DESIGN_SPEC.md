# Empire Industrial Corp - Landing Page 设计规格

## 1. 核心叙事架构 (Narrative Logic)

### 用户心智路径 (User Mental Journey)
```
问题意识 → 发现解决方案 → 建立信任 → 采取行动
"我需要可靠的化工原料供应商" → "他们有专业背景和完整能力" → "他们值得信赖" → "发送询价"
```

### 叙事层次

| 层级 | 内容 | 目的 |
|------|------|------|
| L1 - 身份 | 工程背景的供应链专家 | 差异化定位：不是普通贸易商 |
| L2 - 能力 | 5大行业 + 4大产品支柱 | 展示业务广度和专业深度 |
| L3 - 信任 | 可持续发展=运营标准 | 建立可信度（不是空洞口号） |
| L4 - 行动 | 询价/联系 | 转化目标 |

## 2. 视觉系统 (Visual System)

### 配色方案
```css
--color-primary: #1B4D3E;      /* 深工业绿 - 代表可持续与农业 */
--color-primary-dark: #0F2E25; /* 更深的绿色 - 用于 hover 状态 */
--color-secondary: #4A5568;     /* 高级灰 - 代表工程、钢铁、可靠 */
--color-secondary-light: #718096;
--color-accent: #FFFFFF;        /* 白色 - 内容可读性 */
--color-background: #F7FAFC;    /* 浅灰背景 */
--color-text: #2D3748;          /* 深灰文字 */
--color-cta: #2F855A;           /* CTA 绿色 */
```

### 字体
- 标题: Inter / Helvetica Neue (Bold, 700)
- 正文: Inter / Roboto (Regular, 400)
- 数据: Roboto Mono (用于规格参数)

### 设计原则
1. **工业美学**: 干净、结构化、数据驱动
2. **高端B2B感**: 专业但不冷漠，可信但不刻板
3. **信息层次清晰**: 用户3秒内能理解核心价值
4. **CTA 明确**: 每个关键区域都有行动引导

## 3. 页面结构与内容逻辑

### 3.1 首页 (Home) - 第一印象

**Hero Section**
- 背景: 港口/集装箱的高质量工业图片
- 主标题: "Beyond Logistics: Delivering Industrial Solutions"
- 副标题: 强调北美-亚洲桥梁定位和工程精度
- 双CTA: "查看产品" (主) + "获取报价" (次)

**价值主张条 (Value Bar)**
三个核心差异点的图标+文字展示：
1. Engineering Foundation - 工程根基
2. Regulatory Compliance - 合规保障
3. End-to-End Execution - 全程执行

**信任指标区**
- 服务行业数量
- 产品类别数量
- 服务地区 (北美 & 亚洲)

---

### 3.2 关于我们 (About) - 建立信任

**核心信息层次**
1. 我们是谁: 工业化学品与材料供应商
2. 我们的不同: 工程/化学技术背景支撑所有运营
3. 我们的承诺: 提供实际、可验证、可持续的工业支持

**视觉呈现**
- 左侧: 简洁文字描述
- 右侧: 工业场景图片或数据图表

### 3.4 产品 (Products) - 核心展示区

**结构**: 4大产品类别，每个类别有子产品列表

**Category A: 肥料 (Fertilizers)**
- 氮肥: Urea, UAN Solution, Ammonium Sulphate
- 磷肥: DAP, MAP
- 钾肥: MOP, SOP
- 复合肥/NPK: 定制配方

**Category B: 硫磺 (Sulphur)**
- 形态: Granular, Pastille, Prilled, Lump
- 规格: 纯度、水分、灰分
- 物流: 散装船或集装箱

**Category C: 石油链产品 (Petroleum-chain)**
- 重点: Petroleum Coke (燃烧/冶金用)
- 其他: NGL/LPG, 炼油副产品

**Category D: 工业无机物 (Industrial Inorganics)**
- 产品: HCl, H2SO4, NaOH, NaClO, ZnCl2, ZnO
- 文档: SDS, COA, 第三方检验 (SGS/Intertek)

**每个类别必须有**: "Request a Quote" CTA

## 4. 交互设计

### CTA 层级
1. **Primary CTA**: 深绿色实心按钮 - "Request a Quote"
2. **Secondary CTA**: 绿色边框按钮 - "View Products", "Learn More"
3. **Text Link**: 带下划线的文字链接

### 滚动体验
- 固定导航栏 (滚动时背景变深)
- 平滑滚动到锚点
- 适度的入场动画 (fade-in)

### 响应式断点
- Desktop: 1280px+
- Tablet: 768px - 1279px
- Mobile: < 768px

## 5. 技术实现

### 技术栈
- HTML5 + Tailwind CSS (静态网站)
- 单页面应用 (SPA 风格，锚点导航)
- 响应式图片 (srcset)

### 性能目标
- 首屏加载 < 3s
- Lighthouse Performance > 90

### 图片资源

- 港口/集装箱
- 化工厂
- 矿山作业
- 绿色农田

## 7. 内容语调指南

### 写作风格
- **专业但易懂**: 使用行业术语但解释清楚
- **事实导向**: 用数据和规格说话，避免空洞宣传
- **信心但不傲慢**: 展示能力但不夸大
- **行动导向**: 每段内容都应引导用户下一步

### 示例
❌ "We are the best chemical supplier in the world"
✅ "Engineering-backed supply chain solutions across North America and Asia"

❌ "Contact us for amazing deals"
✅ "Request a quote with your specifications"
